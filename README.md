## Tech Stack

This project was built using the following technologies and tools:

- Frontend:
  - HTML
  - CSS
  - JavaScript
  - ReactJS
  - React Router Dom
  - Tailwind css
  

- Backend:
  - Expressjs
  - Nodejs
  - MongoDB
  - Mongoose
  - Express Validator

- Authentication:
  - JSON web token (jwt)
  
- Password encription
  - Bcryptjs

- Hosting & Deployment:
  - Vercel

- Version Control:
  - Git
  - GitHub
